## 코드 저장소 서비스

Git 저장소를 원격으로 보관하고 협업할 수 있게 해주는 개발 플랫폼

프로젝트의 변경 이력, 브랜치, 커밋을 관리하고 다른 사람과 코드를 공유할 때 사용한다.

## 배포 플랫폼

GitHub Pages를 통해 정적 웹사이트를 무료로 공개할 수 있는 호스팅 수단

이 블로그에서는 Astro가 만든 `dist` 결과물을 웹에 공개하는 배포 위치로 쓰인다.
