## DTO(Data Transfer Object)

계층이나 시스템 사이에서 데이터를 전달하기 위해 사용하는 객체이다.

서버 API 응답을 표현하는 Response 모델 등이 DTO에 해당하며, 외부 데이터 형식과 애플리케이션의 Domain Model을 분리할 때 서로 변환하여 사용한다.
